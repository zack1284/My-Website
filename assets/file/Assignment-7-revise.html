Wine Data Analysis

Overview

The wine dataset includes many characteristics about each type of wine. Each type of wine will be given a quality from 1 to 8. In this analysis, we will classify wine that has quality greater than 6 as high quality wine, otherwise, low quality wine. We will give high quality wine 1, low quality wine 0. Next, we will split the data into training and test set. We will use training set to build models, and utilize test set to predict whether a wine is high quality or not. In addition, we will be using 0.5 as our threshhold.

Preparing Data

In this section, our wine quality has 3-8, 6 kinds of different grades. We define wine having quaility bigger than 6 is high quality wine. We give high quality wine 1, the rest 0.

We split 70% of the data into training set, and the rest as test set.

Logistic Regression

We fit training data to a logistic regression, and use test set to make predictions. We then calculate TPR,TNR for training set and test set.Since we run this before, the result was hidden.

glm <- glm(quality ~ ., family="binomial", data=train)
yhat_glm <- predict(glm, type="response")
test_yhat.glm <- predict(glm, test, type="response")

True Positive Rate (Training Set)

TPR <- function(y,yhat)  { sum(y==1 & yhat==1) / sum(y==1) }
TPR(train$quality, (yhat_glm >0.5))

True Negative Rate (Training Set)

TNR <- function(y,yhat)  { sum(y==0 & yhat==0) / sum(y==0) }

Classification Tree

We use rpart to generate classification tree with training data.

Multi-Layer Network

First, we reformat our factor variables as dummy variables using model.matrix. Then, we scale our data. After data preparation, we use training set to fit our model, we run a two-layered network with 3 and 2 neurons in first and second layer. After training our model, we use test set to generate predictions. We create a confusion table with 0.5 threshold and plot the ROC curve. Our AUC is 0.8712, which performs better than our single layer model, but it’s not better than random forest.

library(neuralnet)
dat4 <- model.matrix( ~ quality + fixed.acidity + volatile.acidity +citric.acid +residual.sugar+chlorides+free.sulfur.dioxide+total.sulfur.dioxide+density +pH+ sulphates+alcohol, data = dat)

train4 <- dat4[trn==TRUE,]
train4 <- scale(train4, center = TRUE, scale = TRUE)
test4 <- dat4[trn==FALSE,]
test4 <- scale(test4, center = TRUE, scale = TRUE)

form4 <- as.formula(quality ~ fixed.acidity + volatile.acidity +citric.acid +residual.sugar+chlorides+free.sulfur.dioxide+total.sulfur.dioxide+density +pH+ sulphates+alcohol)
n4 <- neuralnet(form4, data=train4, hidden = c(3,2), linear.output=FALSE)
n4.test <- model.matrix( ~ fixed.acidity + volatile.acidity +citric.acid +residual.sugar+chlorides+free.sulfur.dioxide+total.sulfur.dioxide+density +pH+ sulphates+alcohol, data = as.data.frame(test4))

AUC Comparison

From the left of the plot is glm, lda, qda, classification tree,random forest, gradient boosting, single layer network, and multi-layer network. In conclusion, random forest performs the best in terms of AUC. Single layer network and multi-layer network did not outperform random forest in this case, but multi-layer network performs slightly better than single layer network.

fit_test <- c(auc(glm.roc),auc(lda.roc),auc(qda.roc),auc(t1.roc),auc(rf1.roc),auc(bt1.roc),auc(n1.roc),auc(n4.roc))
