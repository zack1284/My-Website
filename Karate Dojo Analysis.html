The Introduction to Workflow

First, we will create a database, warehouses, tables in Snowflake. Next, we will pull the data stored in Amazon S3 Bucket into our Snowflake database. We will compute and execute SQL queries in Snowflake. Finally, we will connect Tableau to Snowflake, and visualize the views we created in Snowflake.

Creating a Database

In snowfkale, we created a database name "Census" for us to store data from S3.

Creating Data Warehouses

We will need warehouses for computation purposes. We will specify which warehouses we use when we create worksheets.

Creating Worksheets

Worksheets are places that we can execute SQL queries. Before we execute queries, we will need to specify which warehouse we are using. We created 3 worksheets

  • Census_setup: This is for creating tables, and importing data from S3. It will use "Compute_wh" as our warehouse.
  • Census_analysis: This is for exploring data. It will use "Compute_wh" as our warehouse.
  • Karate Dojo Analysis: This is for karate dojo analysis. It will use "Analytics_wh" as our warehouse.

Importing Data

We will be using the worksheet name "census_setup" to create tables and import data.

In database "census", We could now see the two tables we created and the data in our database. Now, we could execute SQL queries to start our analysis.

Analysis

We will use worksheet named "karate dojo analysis" to perform analysis. Once we find what we want to visualize, we could create views and use Tableau to connect with views.
